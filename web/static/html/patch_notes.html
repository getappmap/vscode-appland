New in this release

Features

  • Remote recording

    Create custom AppMap recordings of your applications all from within Visual Studio Code. Click the record button in the top right of the AppMap sidebar panel to begin a recording. Or, run AppMap: Start remote recording from the command palette.

    Once recording, interact with your application! AppMap will capture the runtime execution until the recording has stopped.

    Stop the recording by clicking the "end recording" button in the top right of the AppMap sidebar panel. Alternatively, run the AppMap: Stop remote recording command from the command palette.

    Use this workflow to supercharge your debugging process, code reviews and breakdown of complex systems.

    Follow the link below for more information:
    https://appland.com/docs/reference/remote-recording

Changes

  • Labels are now displayed as a tag cloud instead of an ordinary list.
  • The details panel will no longer display an empty list. If no data is available for a particular list, it'll instead be hidden.

Bug fixes

  • The tab buttons to change the view will no longer wrap text in smaller viewports.
  • Submitting an empty search from the details panel will no longer cause the view to disappear.
